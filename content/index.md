This is my personal space for notes, ideas, and knowledge.

## Using Quartz

This garden is built using [Quartz](https://quartz.jzhao.xyz/).
It automatically publishes my Obsidian vault as a website.

### Features

- **[[Example Note]]**: Check out how links work!
- **Graph View**: See connections between notes on the right.
- **Search**: Press `Cmd + K` to search.

## Getting Started

1. Open this folder in **Obsidian**.
2. Start writing in the `content` folder.
3. Run `npx quartz serve` to preview changes locally.
